1. The source files, the header files and the makefile is available in the folder “cse522-navneet-jha”.
2. Running the make command generates the library “libuthread.a” along with some “.o” files.
3. Running the make clean command removes the library and the “.o files”.
3. For compiling the test applications, the library needs to be copied to the folder of the test applications, and the test applications need to be compiled with the library.
